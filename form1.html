O atributo type="text" especifica o valor que o input recebe

type="button" cria um botão para um evento

type="checkbox" é usado para dar mais de uma opção para os usuários

type="date" gera um calendário que facilita a inserção da data

type="email" identifica se o campo possui "@" e "."

type="file" abre uma janela para o envio de arquivo.

type="number" determina que o input receba um valor numérico

type="password" gera uma proteção no campo que impede que a senha seja vista

type="radio" é usado para dar mais de uma opção para o usuário

type="range" cria uma barra de progresso

type="search" cria uma barra de pesquisa

type="tel" cria um campo de telefone

type="submit" cria um botão para enviar o formulário.
